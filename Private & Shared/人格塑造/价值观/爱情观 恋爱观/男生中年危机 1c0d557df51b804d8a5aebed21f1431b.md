# 男生中年危机

视频作者讲述母亲下楼时脚崴了，却因舍不得花钱拒绝去医院拍 CT，还提及自己年轻时崴得更严重。作者无奈前往医院买药，在途中感慨老年人节俭，认为若生活富足就不会如此。这件事让作者感受到中年危机，意识到中年危机本质是抵御风险能力不足，虽然在社会中可能不起眼，但作为家庭顶梁柱不能被生活 “拿捏”
