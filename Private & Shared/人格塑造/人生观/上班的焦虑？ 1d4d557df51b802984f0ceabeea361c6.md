# 上班的焦虑？

https://www.bilibili.com/video/BV11v6aY6E1z/?spm_id_from=333.1245.0.0&vd_source=929c876c6676cf0b1cd5c5195a6f28be

三种疲劳根源：
1、想“上班很累每天都在当牛马”，过于抗拒；
2、杂念太多，思考太多，消耗能量；
3、对未来的负面思考。
两种方法改善感知：
1、避免负面词汇，避免无意义思考意义；
2、控制负面情绪，改成积极描述。

这点我个人倒是没有什么意识到。 

1.避免自怨自艾，让负面情绪占满大脑
2.上一天班！=累死了
下班时间，我就应该做自己喜欢的事情，不是虚度光阴。
不然，我会一直刷手机，不睡觉。因为感觉一天就空虚的度过，熬夜，是不愿意承认度过了失败的一天。

这个，我有一点感觉。有时候玩手机也是这样。
